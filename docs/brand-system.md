# SpriteCraft Brand System

SpriteCraft uses a compact brand mark built around three ideas:

- a framed tile to represent spritesheet craft and export structure
- a central spark to represent creative assembly and AI-assisted guidance
- side pixels to hint at frame grids, tools, and game-ready output

## Core mark

The primary Studio mark is:

- a dark rounded square field
- a lighter framed tile
- an orange-to-gold spark at the center
- blue grid pixels at the left and right edges

This keeps the mark readable at favicon size while still feeling specific to sprite tooling instead of generic app chrome.

## Color direction

Primary brand accents:

- ink: `#1f1f28`
- line: `#8ea4d2`
- flame: `#ff8f3f`
- gold: `#f2c56b`

These sit comfortably inside the existing Kanagawa-leaning Studio palette while giving SpriteCraft a warmer, more product-specific focal color.

## Usage guidance

- Use the full wordmark in page chrome and docs where space allows.
- Use the icon-only mark for favicons and compact launch surfaces.
- Keep the spark legible at small sizes; do not replace it with letters unless a platform forces a text-only treatment.
- Prefer the existing Studio palette around the brand mark rather than introducing a second unrelated theme.
