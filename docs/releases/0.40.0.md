# SpriteCraft 0.40.0

## Highlights

- Added a shared Studio toast system for important success, warning, and failure feedback.
- Added compact activity strips to the builder, project browser, and settings page so background work stays visible while the Dart backend is busy.
- Improved feedback around renders, exports, AI actions, imports, project sync, and support-bundle creation.

## Migration notes

- No data or API migration is required for this release.
- Existing SpriteCraft projects, exports, history records, and packaged builds remain compatible.
