# SpriteCraft 0.36.0

Release date: 2026-03-30

## Summary

SpriteCraft 0.36.0 adds first-run onboarding to the Studio overview. New local installs now get a small setup checklist for LPC assets, `.env` setup, and optional Gemini configuration before jumping into the builder.

## Included in this release

- backend-owned onboarding checklist in the bootstrap payload
- Studio overview onboarding panel for first-run and setup-recovery states
- clearer runtime distinction between missing `.env` files and `.env` parsing warnings

## Migration notes

If you are upgrading from 0.35.0:

- no data migration is required
- packaged and repo-based runs will now expose onboarding guidance from runtime bootstrap state
- if you already have a healthy local setup, the onboarding panel will stay out of the way once the setup state is clean and recent work exists

## Release files

Portable Windows bundles now default to a versioned output directory:

```text
build/release/windows/SpriteCraft-win-x64-0.36.0
```
