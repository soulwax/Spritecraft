# SpriteCraft 0.38.0

## Summary

SpriteCraft 0.38.0 adds backend-owned diagnostics packaging for local support work.

This release completes the remaining Phase 8 diagnostics item by adding:

- crash-safe structured log files under `build/logs`
- support bundle zip export under `build/support`
- a Studio Settings action for exporting a support bundle with an optional user note

## What changed

### Structured logging

- Structured backend logs now append to local JSONL files
- Logs continue to write to the terminal, but they now also survive process exits
- The runtime settings summary now exposes the logs directory

### Support bundles

- Added a backend route for creating support bundles
- Added a support bundle panel to Studio Settings
- Support bundles include:
  - `bootstrap.json`
  - `health.json`
  - `manifest.json`
  - recent structured logs
  - export/history recovery indexes

## Migration notes

If you are upgrading from 0.37.0:

- no schema migration is required
- no project migration is required
- new diagnostics artifacts are written under:
  - `build/logs`
  - `build/support`

## Packaging note

Packaged Windows builds should now include this release as:

```text
build/release/windows/SpriteCraft-win-x64-0.38.0
```
