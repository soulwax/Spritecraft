# SpriteCraft 0.39.0

## Summary

SpriteCraft 0.39.0 adds the first cohesive brand layer for the product.

This release completes the first Phase 9 item by introducing:

- a reusable SpriteCraft brand mark and wordmark
- a new SpriteCraft favicon for Studio
- a small brand-system document to keep future polish work aligned

## What changed

### Studio chrome

- Replaced the plain `SC` shell badge with a dedicated SpriteCraft mark
- Updated the Studio metadata description and icon references

### Brand assets

- Added a new `favicon.svg` for Studio
- Added brand-token support in the Studio CSS variables
- Added a shared brand-system document in `docs/brand-system.md`

## Migration notes

If you are upgrading from 0.38.0:

- no runtime migration is required
- no schema migration is required
- existing packaged runs will pick up the new favicon and shell branding on the next web build

## Packaging note

Packaged Windows builds should now include this release as:

```text
build/release/windows/SpriteCraft-win-x64-0.39.0
```
